HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • After editing the style.css sheet, make sure .o link it to the index.html right below the last "meta" element under "head""
  • The "box-shadow" CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas.
  • We can assign colors in CSS in many ways by including semantic references such as "gray" or "blue," or we can also refer to a color by its hexadecimal code, or hex #

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is simply a named container that allows us to store data in our code
  • In JavaScript, to declare a variable, we use the "var" keyword followed by giving the variable a name
  • The ".js" file extension indicates that this file is a JavaScript file
  • Create a new feature branch for the JS file by entering "git checkout -b feature/add-js" into the command line
  • Create a new file inside the assets folder called "script.js" by entering "touch script.js" in the command line
  • Each line of JavaScript code ends with a semicolon (;)
  • the "console.log" method is used to output a message to the web console by adding an argument; "console.log" will output the value of "topics" to the web console because we've added it as an argument inside the parentheses
  • To link the "script.js" file to the "index.html" file, add the code under the "footer" element and above the closing element for "body"
  • In order to see changes that we made with JavaScript, we need to use Chrome DevTools
  • Order of the code affects the outcome since JavaScript is read from top to bottom
  • A value is considered "falsy" if it is: "0," "-0," "On," " ," "null," "undefine," "Nan," or a Boolean data type value with the value of false"
  • An array is a single variable that is used to hold a group of data and are typically used to hold data that is related in some way
  • All other values, including a Boolean witht he value of "true," are considered "truthy"
